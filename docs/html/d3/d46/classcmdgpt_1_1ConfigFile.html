Detailed Description

Configuration file manager.

Handles loading and saving configuration from/to .cmdgptrc files. Supports simple key=value format with comment lines starting with #.

ConfigFile config;
if (config.load(ConfigFile::get_default_path())) {
config.apply_to(my_config);
}

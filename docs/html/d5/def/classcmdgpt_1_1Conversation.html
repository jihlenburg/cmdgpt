Detailed Description

Manages conversation history and context.

The Conversation class maintains a history of messages exchanged with the AI model, automatically managing context length to stay within token limits. It provides persistence through JSON serialization and file I/O.

Conversation conv;
conv.add_message("system", "You are a helpful assistant");
conv.add_message("user", "Hello!");
conv.save_to_file("conversation.json");
